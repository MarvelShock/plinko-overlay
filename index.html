📺 Add as OBS Browser Overlay

  1. In OBS, click + under Sources → choose Browser
  2. Set the URL to:
https://marvelshock.github.io/plinko-overlay/
  1. Set Width to your canvas width (e.g. 640)
  2. Set Height to your canvas height (e.g. 560)
  3. Check "Shutdown source when not visible" to save resources
  4. Click OK — position and resize the source as needed

💡 Twitch chat auto-connects when the page loads — no extra setup needed. Only tyler25 and thenicolet can trigger drops via !drop in chat.
